Installation process

 

1 - If you don't already have a RoF2 client copy, you can get one of the server's ready to play Imperium RoF2 EverQuest client here.

 

2 - If you already have a copy of the RoF2 client, you will need to download the following server files:

 

    (do not forget to backup the original files just in case)

 

   - dbstr_us file  (goes into the main installation folder, replace the original)

 

   - spell_us file (goes into the main installation folder, replace the original)

 

   - GlobalLoad file (goes into the resources folder, replace the original)

 

   - Maps Files (drop into the main installation folder, replace everything if asked)

 

 

3 - Change your eqhost.txt file with the following:

 

      [LoginServer]

 

      Host=login.eqemulator.net:5999

 

 

4 - Creating a shortcut launcher executable; This will bypass the EQLive patcher. First, locate the eqgame.exe file in the main installation folder. Right-click and select "Create Shortcut". You can rename it and drag it anywhere on your desktop.

 

Now, right-click that shortcut and select "Properties" In the target box, you will see the path to the eqgame.exe file. We need to edit this in order to bypass the patcher.

 

At the end of the text box (after any parentheses), add "patchme" (no quotes). Your target box should look similar to the image on the right.

 

5 - Once you've created a shortcut with the instructions above, you will be able to finally connect with your EQemulator account, select the server and play!

 

 Otherwise, if you do not already have an EQemulator account;

 

- Make a Forum account first.

 

- Once you've made one, create a Loginserver account, located on the left side bar on the EQemulator homepage when logged in.

 

Once you're all done, log into your emulated version of Everquest, select your desired server to play on and have fun! (The Imperium server will be located in the green area "Preferred Servers")
